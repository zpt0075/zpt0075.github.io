第一次搭建博客成功

Posted at 2025-02-05 感想 

纪念一下，激动的心难以压制！！！

Hexo + GitHub Pages + Cloudflare Pages

这个活字印刷的主题是我从关注的一个B站up主polebug32哪里知道的，觉得挺不错，就用了同款。
